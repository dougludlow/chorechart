# Chore Chart API

An API for managing household chores.

## Getting Started

### Prerequisites 

- Install [.NET Core](https://www.microsoft.com/net/download)

### Building

```bash
make build
```

### Running

```bash
make
```

## Testing

```bash
make test
```
